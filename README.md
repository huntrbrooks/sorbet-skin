# Sorbet Skin

Sorbet Skin is an original interactive e-commerce demo for a playful, premium self-tanning bodycare range.

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite

## Local Setup

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Notes

This is a frontend-only demo. Cart state uses `localStorage`, and checkout is mocked for demonstration purposes.
